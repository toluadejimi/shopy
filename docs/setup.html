Queue - Schedule Task (Supervisor)

  1. Run sudo apt-get install supervisor to install supervisor in ubuntu

  2. Create laravel-worker.conf in /etc/supervisor/conf.d/
    or Run vi /etc/supervisor/conf.d/laravel-worker.conf

  3. Download this file & Copy contains and paste to laravel-worker.conf

  4. You need to change project location in this file to your location

  5. Run following command to activate supervisor for this file

    • sudo supervisorctl reread

    • sudo supervisorctl update

    • sudo supervisorctl start laravel-worker:*

    • sudo supervisorctl status laravel-worker:*

How to Run (Locally)

Note: Run all command at laravel root project

  1. Local Server - Optional

    • Run php artisan serve to serve over your localhost

    • Hit http://localhost:8000/admin on browser to view admin panel

  2. On Your IP - Recommended

    • You need to find your local ip (IPv4). You can try a here

    • Run php artisan serve --host={IP}

      Example: If your ip is = 192.168.1.74 then you need to run php artisan serve --host=192.168.1.74

  3. Queue - Schedule Task

    • Run php artisan queue:work --queue=high,default to start workers to work async tasks

Storage

Run php artisan storage:link to link storage to public

If you need to upload files more than 2 MB (megabytes). Then follow steps to increase post request size

  1. Open file /etc/php/{PHP_VERSION}/cli/php.ini

    Example: If you use php 8.1 then open /etc/php/8.1/cli/php.ini

  2. Set following values to whatever you need to be max

    post_max_size = 6G
    upload_max_filesize = 4G
    memory_limit = 8G

  3. Save file as root/administrator
