Prerequisites

  1. Node v14+ (If you haven't than install from https://nodejs.org/en/download/)

  2. Yarn (If you haven't than install from https://classic.yarnpkg.com/lang/en/docs/install/)

How to compile web

  1. Open terminal/command prompt at root location of laravel project

  2. Run yarn command

  3. Then yarn watch command to compile assets and watch over the assets

  4. For Production: Run yarn production to compile all assets to production ready

Firebase Setup on Web

  • You need to setup firebase Here . Then follow these steps

  • In firebase console, Add app & select web

  • Set app name whatever you want & click on "Register app"

  • Copy const firebaseConfig = { ... } part and paste following places

    • public/firebase-messaging-sw.js

    • resources/js/firebase.ts

  • Paste your config to this 2 place & It's done
