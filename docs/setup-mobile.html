Prerequisites

  1. Flutter 3.3.x (https://docs.flutter.dev/get-started/install)

  2. Android Studio (https://developer.android.com/studio/)

  3. Run flutter doctor command to confirm flutter version.

How to run app

  1. From Android Studio

    • Open flutter project to android studio & Select emulator or connect your device (physical)

    • Click on Run button on top-right

  2. From Terminal/Command prompt

    1. Open terminal at any flutter project(app)

    2. Run flutter pub get to getting all dependencies, related to that project

    3. Run flutter run to run app on device [Flutter show you a devices list, If you connected more than 1]

Firebase Setup on Mobile

  • You need to setup firebase Here . Then follow these steps

  • In firebase console, Add app & select android

  • Set android package name, which find from android/app/src/main/AndroidManifest.xml

  • Set Debug signing certificate SHA-1. How to find. It's optional

  • Click on Register app

  • Download google-services.json and paste this file to android/app/google-services.json

  • It's done

Deep Link (Referral Link)

  • Go to android/app/src/main/AndroidManifest.xml

  • Change android:host= to your website url

  • Go to lib/app_setting.dart

  • Set deepLinkBaseURL to your website url lib/app_setting.dart
